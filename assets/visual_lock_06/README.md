# Visual Lock 6 Source Art Pack

This folder holds benchmark-only object placeholder source art for Visual Lock 6. Terrain source
overrides live in `assets/sprite_styles/cozy_upland_sparse/overrides/` because the sparse Ridge
Trap benchmark uses that SpriteGen style profile.

The object PNGs here are copied into `exports/visual_lock_06/benchmark_override_pack/object_placeholders/`
when `visual-lock-benchmark` runs. They document the current replacement targets for tree, log,
rock, and objective marker art; object sprite manifests are still a later art-kit integration step.
